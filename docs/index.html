Machine Setup

Cross-platform machine configuration in pure bash. Profiles, dotfiles, backups — one command.

$ curl -fsSL https://raw.githubusercontent.com/gregnazario/machine-setup/main/setup.sh | bash
Get Started View on GitHub

Features

Everything you need to set up and maintain machines, with nothing you don't.

🌐

18+ Platforms

Fedora, Ubuntu, Arch, macOS, FreeBSD, Windows, NixOS, Termux, and more. One tool handles them all with platform-aware package mapping.

📝

Profile System

Choose minimal, full, selfhosted, or community profiles. Create custom ones with INI-based inheritance — no YAML, no JSON, no dependencies.

📄

Dotfile Sync

Symlink-based dotfile management organized by profile. Real-time sync across machines via Syncthing. Encrypted secrets with git-crypt.

🔒

Encrypted Backups

Daily automated backups with Restic to BackBlaze B2 or S3. Verification, restore, and rotation built in.

🚀

Fleet Management

Register machines, assign profiles, and run setup across your entire fleet via SSH from a single repo.

⚙

Zero Dependencies

Pure bash with a built-in INI parser. No Ruby, Python, Node, or Ansible required. Just curl, git, and a shell.

Quick Start

Up and running in three steps.

1

Install

Clone the repo or run the one-liner to bootstrap automatically.

git clone https://github.com/gregnazario/machine-setup.git
cd machine-setup
2

Pick a Profile

Choose what kind of setup you need.

# Servers & minimal environments
./setup.sh --profile minimal

# Full dev workstation
./setup.sh --profile full

# Self-hosted services
./setup.sh --profile selfhosted
3

Verify

Check that everything was set up correctly.

# Health check
./setup.sh --check --profile full

# Status dashboard
./setup.sh --status
